\pregunta[20]
\textbf{Código:} Analice el funcionamiento de la siguiente función en Python 3:
\begin{minted}[linenos, breaklines]{python}
    def fib(n):
        a, b = 0, 1
        while a < n:
            print(a, end=' ')
            a, b = b, a + b
        print()

    fib(1000)
    \end{minted}
